الكود الأصلي

print("🎯 Welcome to the Guessing Game!")
secret_number = 7

guess = int(input("Guess a number between 1 and 10: "))

if guess == secret_number:
    print("🎉 Wow! You guessed it right!")
elif guess > secret_number:
    print("Too high! Try a smaller number.")
else:
    print("Too low! Try a bigger number.")
السطر 1: print("🎯 Welcome to the Guessing Game!")

يطبع رسالة ترحيب للاعب باللغة الإنجليزية.

السطر 2: secret_number = 7

يحفظ الرقم السري الذي يجب على اللاعب تخمينه.

السطر 3: guess = int(input("Guess a number between 1 and 10: "))

يطلب من اللاعب إدخال رقم بين 1 و 10، ويحوله إلى عدد صحيح.

السطر 4: if guess == secret_number:

يتأكد إذا كان الرقم الذي أدخله اللاعب يساوي الرقم السري.

السطر 5: print("🎉 Wow! You guessed it right!")

يطبع رسالة الفوز إذا التخمين صحيح.

السطر 6: elif guess > secret_number:

يتأكد إذا الرقم الذي أدخله اللاعب أكبر من الرقم السري.

السطر 7: print("Too high! Try a smaller number.")

يطبع رسالة تقول أن الرقم كبير جداً.

السطر 8: else:

ينفذ إذا كان الرقم أصغر من الرقم السري.

السطر 9: print("Too low! Try a bigger number.")

يطبع رسالة تقول أن الرقم صغير جداً.

📝 التمارين

تمرين 1: غير الرسالة في السطر 1 لتقول "Hello in Guessing Game!"
تمرين 2: غير الرقم السري ليكون 5.
تمرين 3: غير النص في إدخال اللاعب ليكون "Enter a number:"
تمرين 4: أضف شرط إذا الرقم أكبر من 10 يطبع "Out of range"
تمرين 5: غير رسالة الفوز لتقول "Great job! You won!"
تمرين 6: أضف بعد شرط الرقم الأكبر طباعة "Try a smaller number"

✅ الحلول

🎮 الكود النهائي

secret_number = 7

guess = prompt("Guess a number between 1 and 10:")

if (parseInt(guess) === secret_number) {
    alert("🎉 Wow! You guessed it right!");
} else if (parseInt(guess) > secret_number) {
    alert("Too high! Try a smaller number.");
} else {
    alert("Too low! Try a bigger number.");
}
